# 21 — Rejected and Deferred Ideas (Hackathon)

Sources: `talk2`, `talk3`, `talk8`, `talk9`, `talk11`, and others.

The core cutting rule is:

> Give up claims that cannot be externally verified, not the concept itself.

> **Chronology constraint:** KILL, deferred, and candidate statuses are branch-
> local judgments. One branch does not necessarily overwrite another in time.
