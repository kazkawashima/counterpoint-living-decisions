# Reel and visual evidence

This directory stores synthetic, submission-safe visual material for the
hackathon introduction reel.

## Layout

- `screenshots/<feature>/`: screenshots captured after UI changes.
- `clips/<feature>/`: short source clips or capture notes for animations and
  interactions.
- `notes.md`: optional reel sequencing, narration, and cut-point notes.

Use stable filenames and include the viewport and state in the accompanying
note. Never capture secrets, private user data, or real decision content.
